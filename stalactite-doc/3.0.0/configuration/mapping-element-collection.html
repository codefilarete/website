Mapping a collection of elements

To map a Collection of elements (not entities) you must use the mapCollection(..) method, which takes as second argument the element type.

MappingEase.entityBuilder(Person.class, Long.class)
    .mapKey(Person::getId, IdentifierPolicy.afterInsert())
    .mapEnum(Person::getNicknames, String.class)

If your element type is complex (more than one column) you shall give the bean configuration as third argument.

MappingEase.entityBuilder(Person.class, Long.class)
    .mapKey(Person::getId, IdentifierPolicy.afterInsert())
    .mapEnum(Person::getTimestamps, Timestamp.class, MappingEase.embeddableBuilder(Timestamp.class)
            .map(Timestamp::getCreationDate)
            .map(Timestamp::getModificationDate))
